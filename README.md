### Overview
FdF is a shcool project. The goal is to create an isometric map viewer.

Implemented features : 
* Input file and parsing error detection
* Zoom
* Move
* Scale
* Auto-centering
* Color

### Controls
* **Q/A**: Scale X +/-
* **W/S**: Scale Y +/-
* **E/D**: Scale Z +/-
* **F**: Toggle auto-centering of center
* **C**: Toggle color
